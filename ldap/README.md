## Usage:
* setup: `docker-compose up -d `
* gui accessed from: **http://localhost:8080/**
* service accessed from: **http://localhost:8090/**
* teardown: `docker-compose down`
* login with:
    * login: `cn=admin,dc=example,dc=org`
    * pass: `admin`
